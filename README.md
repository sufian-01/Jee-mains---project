An end-to-end Data Analytics project analyzing multi-year JEE admission cutoff data using **Python (Pandas)** and **Power BI**.

 Project Overview

This project analyzes **64,958 admission records** across **14 structured columns**, covering JEE admission cutoffs from **2016 to 2021**.

The dataset includes:
- Institute Type (IIT / NIT)
- Program Name
- Category (GEN, OBC-NCL, SC, ST, PWD, etc.)
- Quota
- Opening Rank
- Closing Rank
- Year

The objective was to uncover competitiveness trends across institutes, programs, and reservation categories.
